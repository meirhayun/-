# חיון הרחקת יונים — אתר

אתר תדמית רב-דפי לחברת הרחקת יונים, בנוי עם **Vite** ו-**TypeScript**.

## דפים

| דף | קובץ | תיאור |
|----|------|-------|
| בית | `index.html` | עמוד ראשי: hero, שירותים, תהליך, אודות, תצוגת ביקורות |
| צור קשר | `contact.html` | טופס יצירת קשר עם ולידציה + פרטי התקשרות |
| ביקורות | `reviews.html` | כל הביקורות, סיכום דירוג וסינון לפי סוג שירות |

## דרישות מקדימות

יש להתקין **Node.js** (גרסה 18 ומעלה) — ניתן להוריד מ-<https://nodejs.org>.

## הרצה

```bash
# התקנת התלויות (פעם אחת)
npm install

# הרצה בסביבת פיתוח (נפתח אוטומטית בדפדפן)
npm run dev

# בניית גרסת ייצור לתיקיית dist/
npm run build

# תצוגה מקדימה של גרסת הייצור
npm run preview
```

## מבנה הפרויקט

```
.
├── index.html              # דף הבית
├── contact.html            # דף יצירת קשר
├── reviews.html            # דף ביקורות
├── styles.css              # עיצוב משותף לכל הדפים
├── src/
│   ├── types.ts            # טיפוסים משותפים (Review, ContactFormData ...)
│   ├── components/
│   │   ├── nav.ts          # תפריט נייד, צל בגלילה, קישור פעיל
│   │   ├── reveal.ts       # אנימציית חשיפה בגלילה
│   │   ├── contactForm.ts  # ולידציה ושליחת טופס
│   │   └── reviews.ts      # סיכום דירוג וסינון ביקורות
│   └── pages/
│       ├── home.ts         # נקודת כניסה לדף הבית
│       ├── contact.ts      # נקודת כניסה לדף יצירת קשר
│       └── reviews.ts      # נקודת כניסה לדף הביקורות
├── vite.config.ts          # הגדרת ריבוי דפים
├── tsconfig.json           # הגדרות TypeScript (מצב strict)
└── package.json
```

## הערות

- ה-TypeScript בנוי כ"שיפור הדרגתי" (progressive enhancement): התוכן מוצג גם ללא JavaScript, וה-TS מוסיף אינטראקטיביות (סינון, ולידציה, אנימציות).
- טופס יצירת הקשר מדמה שליחה (מדפיס ל-console). לחיבור אמיתי יש להוסיף קריאה ל-API או לשירות דיוור ב-`src/components/contactForm.ts`.
